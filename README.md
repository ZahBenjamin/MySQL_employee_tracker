  ## Installation

  To install first run command 'npm init -y', then 'npm install' for inquirer, console.table, and MySQL.

  ## Usage

  For seeding the DB:
  ```
  mysql -u root
  SOURCE db/schema.sql;
  SOURCE db/seeds.sql;
  ```

  To start the file:
  ```
  node server.js 
  ```
